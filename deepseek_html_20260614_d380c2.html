Guide d'utilisation

🔹 Coûts variables : Ajoutez/modifiez les postes (ex: ciment, gazole). Le coût unitaire se calcule automatiquement.

🔹 Charges fixes : Loyer, salaires, etc. Le seuil de rentabilité s'ajuste en temps réel.

🔹 KPI : CA, marge brute, taux, seuil CA, score de pilotage (sur 100).

🔹 Graphique : Courbes CA et coûts totaux. Le point d'intersection = seuil de rentabilité.

🔹 Optimiseur prix : Calcule un prix recommandé basé sur la distance et les coûts.

🔹 Intégrations : Configurez vos URLs n8n, webhook, API Python. Testez les connexions. Les alertes peuvent être envoyées automatiquement (bouton "Envoyer alerte" dans le ticket).

🔹 Sauvegarde : Tous les paramètres sont sauvegardés automatiquement dans votre navigateur.

🔹 Dark mode / Personnalisation : Cliquez sur l'icône ⚙️ pour changer le nom, unités, couleurs.
